As per the POLYMAZE 2026 competition, me and my team have built a maze solver robot,
the robot, as allowed in the competition, navigates through the maze by following a black line
thus, the robot's code consists of two parts : Line following, and the maze solving algorithm
we opted for the right hand rule method, the robot solves the maze by following it's right, and thus by ignoring left paths in intersections

MATERIALS USED :
1- QTR SENSOR (ANALOG) 8-array qtr
2- L298N Motor driver
3- 2* N20 motors (12v, 1000 RPM)
4- ESP32
5- breadboard + jumper wires
6- 3D printed frame.
8- Circle of beads
